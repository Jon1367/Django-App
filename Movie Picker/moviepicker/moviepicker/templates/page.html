Next Were going to setup the Templating

//App Sets
app.set('views', __dirname);
app.set('view engine', 'ejs');


This is a necessary step to tell node.js where your templates are. Also you’re going to want to get embed.js (ejs) to display information to the dom. You can also use jade. The view engine tells node.js what to look for. Once that’s complete you can step up your first route

Rendering Templates in your App.js

// Includes
var express = require('express');
var app = express();
var http = require('http');
var path = require('path');
var ejs = require('ejs');

//App Sets
app.set('views', __dirname);
app.set('view engine', 'ejs');

// Routes
app.get('/login', function(req, res) {
res.render('./views/form');
});

app.listen(8080);
console.log("Listening on port 8080");

To render a template in node.js you’ll need to first use the app.get that we use previous. This time instead of using res.send(‘ hello’) your going to use res.render to render a template of your choice. Ex. res.render('./views/form');. go to localhost:8080/login to view your template.
